## Getting Started

<Steps>
<Step title="🔑 Get Your API Access">
Sign up for a Postreach AI account and obtain your session credentials from the dashboard.

<Card title="Create Account" icon="user-plus" href="https://dashboard.postreach.ai/signup">
  Get started with our free tier - no credit card required
</Card>
</Step>

<Step title="📖 Explore the API">
Familiarize yourself with our comprehensive API endpoints and capabilities.

<CodeGroup>
```bash Quick Test
curl -X POST 'https://api.postreach.ai/pipeline/generate' \
  -H 'Content-Type: application/json' \
  -H 'X-Session-ID: your-session-id' \
  -d '{
    "mode": "Autopilot",
    "social_post_number": {"linkedin": 1, "twitter": 2},
    "user_info": [{"type": "topic", "content": "AI trends 2024"}],
    "schedule_start_date": "2024-01-20T09:00:00Z",
    "schedule_end_date": "2024-01-27T18:00:00Z"
  }'
```

```python Python
import requests

response = requests.post('https://api.postreach.ai/pipeline/generate', 
  headers={'X-Session-ID': 'your-session-id'},
  json={
    'mode': 'Autopilot',
    'social_post_number': {'linkedin': 1, 'twitter': 2},
    'user_info': [{'type': 'topic', 'content': 'AI trends 2024'}],
    'schedule_start_date': '2024-01-20T09:00:00Z',
    'schedule_end_date': '2024-01-27T18:00:00Z'
  })
```

```javascript JavaScript
const response = await fetch('https://api.postreach.ai/pipeline/generate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Session-ID': 'your-session-id'
  },
  body: JSON.stringify({
    mode: 'Autopilot',
    social_post_number: {linkedin: 1, twitter: 2},
    user_info: [{type: 'topic', content: 'AI trends 2024'}],
    schedule_start_date: '2024-01-20T09:00:00Z',
    schedule_end_date: '2024-01-27T18:00:00Z'
  })
});
```
</CodeGroup>
</Step>
